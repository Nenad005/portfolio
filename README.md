# Portfolio

Kratak portfolio projekat sa React frontend-om i Strapi backend-om.

## Tehnologije

- React
- Strapi
- GraphQL / Apollo Client
- Tailwind CSS
- React Scroll, React CountUp i Typewriter efekti

## Pokretanje

Frontend:

```bash
npm install
npm start
```

Backend:

```bash
cd backend
npm install
npm run develop
```

## Napomena

Ovaj projekat je outdated i nije aktivno odrzan setup
Backend može da se aktivira tek nekoliko minuta nakon prvog request-a, pa je normalno ako inicijalni poziv kasni ili prvo vraća prazniji odgovor.
